## ✨ Features

### 🖱️ **Custom Plane Cursor**
Tired of boring default cursors? Me too! That's why I built a custom animated paper airplane cursor that follows your movements with smooth physics-based easing. It even changes color based on which section you're hovering over!

### 🃏 **3D Flip Cards**
Each project card flips in 3D space to reveal detailed information on the back—complete with problem statements, solutions, results, and metrics. No more clicking through multiple pages!

### 🎯 **Smooth Scroll Navigation**
Buttery smooth scrolling between sections with active link highlighting. The navbar stays sticky so you can navigate anywhere instantly.

### 🌓 **Dark/Light Mode**
Toggle between light and dark themes with a single click. Your preference is saved in localStorage, so it remembers your choice on return visits.

### 🎮 **Playable Arcade Game**
Yes, there's a fully playable **Space Defender** game built right into the portfolio! 20+ levels, boss fights, power-ups, and local high scores. Take a break from browsing and shoot some aliens!

### ✨ **Particle Effects**
Hover over project cards to trigger beautiful particle explosions. The particles match each project's accent color for that extra touch of polish.

### 🌀 **Animated Background**
Subtle floating blobs and a gradient background create depth and visual interest without distracting from the content.

### 📱 **Fully Responsive**
Looks gorgeous on everything from 4K monitors to mobile phones. The layout adapts seamlessly to any screen size.

### 🔍 **Project Filtering**
Filter projects by category (Web Apps, AI Tools, Games, Business) to quickly find what you're interested in.

### 📊 **Real Metrics, Not Fluff**
Each project includes actual metrics—GitHub stars, performance scores, user counts—because numbers speak louder than words.

### ⚡ **Vite-Powered**
Blazing fast development and build times thanks to Vite. Hot module replacement makes development a joy.

## 🎨 Design Philosophy

### **"Code Should Feel Alive"**

I believe that great websites aren't just functional—they're experiences. Every animation, every transition, every hover effect in this portfolio serves a purpose:

1. **Delight without distraction** - The plane cursor is fun but subtle. The particle effects reward interaction without overwhelming.
2. **Information hierarchy** - Project cards show the most important info first, with details revealed on hover.
3. **Consistent visual language** - The purple/cyan gradient theme ties everything together.
4. **Accessibility first** - High contrast ratios, proper heading structure, and keyboard navigability.

### **Color Palette**
```css
--primary: #6c5ce7;    /* Purple - energy, creativity */

--bg-dark: #0a0c14;     /* Deep space */

```css
- perspective: 2000px for realistic depth
- transform-style: preserve-3d for proper 3D

## 🔧 Installation & Setup

Want to run this locally or use it as a template? Here's how:

### Prerequisites
- Node.js (v18 or higher)
- npm (v9 or higher)
- Git

### Step-by-Step Setup

1. **Clone the repository**
```bash
git clone https://github.com/Dani1157/my-portfolio.git
cd my-portfolio
```

2. **Install dependencies**
```bash
npm install
```

3. **Run the development server**
```bash
npm run dev
```

4. **Open your browser**
```
http://localhost:5173/my-portfolio/
```

5. **Build for production**
```bash
npm run build
```

6. **Preview the production build**
```bash
npm run preview
```

## 🌐 Deployment

### Deploying to GitHub Pages

This portfolio is configured for easy deployment to GitHub Pages:

1. **Update the base path** in `vite.config.js`:
```javascript
export default defineConfig({
  base: '/my-portfolio/',
  plugins: [tailwindcss()]
})
```

2. **Build the project**
```bash
npm run build
```

3. **Deploy to gh-pages branch**
```bash
git add dist -f
git commit -m "Deploy to gh-pages"
git subtree push --prefix dist origin gh-pages
```

4. **Enable GitHub Pages**
- Go to repo Settings → Pages
- Select `gh-pages` branch
- Root folder `/`
- Save and wait 2-3 minutes
